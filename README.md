# 🔐 Advanced Network Port Scanner

> ⚡ Scan. Analyze. Secure.

A modern GUI-based **multithreaded network port scanner** with real-time analysis, risk classification, and automated security report generation.

---

## 🚀 Features

* ⚡ Fast multithreaded port scanning
* 🎨 Modern dark-themed GUI (Tkinter)
* 🧠 Risk classification (HIGH / MEDIUM / LOW)
* 📄 Professional security report generation
* 📁 Export reports (TXT & PDF)
* 📊 Risk distribution graph (matplotlib)
* 🛑 Start / Stop control

---

## 🖥️ Tech Stack

* Python 3
* Tkinter (GUI)
* Socket Programming
* ThreadPoolExecutor
* Matplotlib
* ReportLab

---

## ▶️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/advanced-port-scanner.git
cd advanced-port-scanner
python -m pip install -r requirements.txt
```

---

## ▶️ Run

```bash
python main.py
```
